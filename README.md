India Sales Performance Dashboard – Excel Project

This project analyzes retail sales data across multiple Indian states, product categories, and months.
Using Excel, the dataset was cleaned, transformed, and visualized to create an interactive, professional sales dashboard.

The project demonstrates skills in:
✔ Data Cleaning
✔ Pivot Tables
✔ KPI Development
✔ Dashboard Design
✔ Data Interpretation
✔ Slicers, Charts, and Map Visualization

1. Data Cleaning Steps

The dataset was cleaned using the following operations:
	•	Removed duplicate rows
	•	Standardized product names
	•	Corrected inconsistent category labels
	•	Converted dates to proper date format
	•	Created separate Month column
	•	Fixed text casing and trimmed spaces
	•	Ensured all numerical columns were in number format
	•	Recalculated Total Sales = Units Sold × Unit Price
